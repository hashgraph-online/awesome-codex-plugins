# Security

To report a security vulnerability in AgentBox, please open an issue at
https://github.com/madarco/agentbox/issues with the `[security]` label, or email the author
via the contact links at https://www.madarco.net.

AgentBox keeps host git credentials on the host (never inside a box); push and other
host-side operations are mediated by a host relay with permission prompts.
