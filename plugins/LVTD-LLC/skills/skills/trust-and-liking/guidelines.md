# Trust and Liking Guidelines

Quick routing for trust and liking tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/liking/knowledge.md |
| Create or rewrite copy | references/liking/rules.md, references/liking/examples.md |
| Audit an existing surface | references/liking/smells.md, references/liking/checklist.md |
| Run the full workflow | workflows/build-rapport-without-flattery.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/liking/smells.md |
| The evidence behind a claim is unclear | references/liking/rules.md |
| The user needs a practical rewrite | references/liking/examples.md |
