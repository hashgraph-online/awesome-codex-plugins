# LinkedIn Post Experimentation Examples

## Hypothesis Examples

```markdown
If we turn a text-only lesson into a document post, saves and sends will rise
because the audience can reuse the checklist.
```

```markdown
If we replace a broad thought-leadership hook with a specific audience problem,
qualified comments will improve even if impressions stay flat.
```

```markdown
If we publish industry news with a "what this means" interpretation within 24
hours, profile visits from the target industry should increase.
```

## Test Plan Template

| Field | Example |
|-------|---------|
| Variable | Format: text post vs document post |
| Audience | B2B content leads |
| Baseline | Last 5 checklist-style posts |
| Metric | Saves, sends, comments from target roles |
| Window | 7 days |
| Repeat Rule | Repeat if saves or qualified comments beat baseline by 25% |

## Decision Examples

- **Repeat**: Same topic and format produced high saves and good comments from
  the right audience.
- **Revise**: Impressions were strong but comments were generic; sharpen the
  audience and question.
- **Stop**: High reach came from irrelevant audience or controversy that does
  not support positioning.
