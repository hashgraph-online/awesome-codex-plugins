# LinkedIn Post Experimentation Rules

## Hypothesis Rules

- Write the audience and variable explicitly.
- Test one main variable at a time.
- Define the expected signal before publishing.
- Choose a comparison: previous similar post, alternate format, alternate hook,
  or baseline cadence.

## Metric Rules

- Prefer qualified signals over vanity metrics.
- Track comments by quality, not only quantity.
- Track saves and sends when the post is meant to be useful.
- Track profile visits or follower growth when the post is meant to build
  personal brand.
- Track newsletter subscriptions when promoting a newsletter.
- Use impressions to understand distribution, but do not treat reach as proof
  of value.

## Timing Rules

- Choose posting windows based on the user's actual audience when known.
- Test timing rather than accepting universal "best time" advice.
- Avoid judging a post before comments and distribution have had time to settle.
- If scheduling posts, make sure the author can respond to comments.

## Analysis Rules

- Compare similar posts before drawing conclusions.
- Separate format performance from topic performance.
- Note outliers: mentions by high-reach people, external promotion, unusual
  news timing, holidays, or platform changes.
- Turn every analysis into a next content hypothesis.

## Safety Rules

- Do not use fake engagement to test content quality.
- Do not buy engagement.
- Do not recommend automation that manipulates LinkedIn.
- Flag outdated book tactics when current LinkedIn policy or features differ.
