# LinkedIn Post Experimentation Knowledge

## Core Idea

The book presents growth hacking as a loop: ideate, prioritize, test, analyze,
then ideate again. Applied to LinkedIn content, each post is a small experiment
about which audience, topic, format, hook, or call to action creates useful
attention and trust.

## Variables Worth Testing

| Variable | Examples |
|----------|----------|
| Topic | Industry news, personal lesson, method, opinion, data, case study. |
| Hook | Problem, contrarian point, story, result, question, surprising implication. |
| Format | Text, image, document, video, article, newsletter, poll, event. |
| Media | Native video, document, image with alt text, screenshot, diagram. |
| Timing | Workday morning, weekend, middle of month, event-driven timing. |
| Discoverability | Hashtags, mentions, topic consistency. |
| Conversation | Question style, provocative but professional claim, reply behavior. |

## Metrics And Signals

| Signal | What It Usually Means |
|--------|-----------------------|
| Impressions | Distribution; useful but can be broad and low-quality. |
| Comments | Conversation and resonance; inspect quality, not only count. |
| Saves | Practical usefulness. |
| Sends | Private shareability. |
| Reposts | Distribution, but may not always produce the best conversation. |
| Follower growth | Topic association and continuing interest. |
| Profile visits | Curiosity about the author. |
| Qualified conversations | Highest-value signal for business or career goals. |

## Experiment Discipline

Good experiments isolate a variable and define the learning before publishing.
The goal is not to "win LinkedIn"; it is to discover which content reliably
serves the audience and supports the author's positioning.
