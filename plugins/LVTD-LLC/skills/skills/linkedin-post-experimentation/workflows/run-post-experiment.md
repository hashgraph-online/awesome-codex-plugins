# Run LinkedIn Post Experiment Workflow

Use this workflow when the user wants to test LinkedIn hooks, formats, cadence,
topics, hashtags, or content performance.

## Steps

### Step 1: State The Hypothesis

- [ ] Audience.
- [ ] Content variable.
- [ ] Expected signal.
- [ ] Business, career, or brand goal.

### Step 2: Design The Test

- [ ] Choose one variable.
- [ ] Define comparison or baseline.
- [ ] Choose format.
- [ ] Choose publishing window.
- [ ] Define comment response plan.

### Step 3: Define Metrics

- [ ] Impressions.
- [ ] Engagements.
- [ ] Comment quality.
- [ ] Saves or sends.
- [ ] Profile visits.
- [ ] Follower or newsletter subscriber change.
- [ ] Qualified conversations.

### Step 4: Analyze

- [ ] Compare against similar posts.
- [ ] Identify confounders.
- [ ] Separate topic, hook, and format effects.
- [ ] Decide repeat, revise, or stop.
- [ ] Write the next hypothesis.

## Exit Criteria

- [ ] The user knows what to publish next and why.
- [ ] The decision is based on audience-fit signals, not vanity reach alone.
- [ ] No fake engagement or platform manipulation is part of the test.
