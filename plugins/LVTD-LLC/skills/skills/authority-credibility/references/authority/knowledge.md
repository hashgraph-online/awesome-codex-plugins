# Authority Knowledge

Source basis: Robert B. Cialdini, Influence: The Psychology of Persuasion. The concepts below are paraphrased for skill use; this file does not require access to the local extraction used during authoring.

## Overview

Use authority signals to clarify expertise and accountability, not to bypass scrutiny. Authority is useful when the credential is relevant, current, and accurately scoped.

## Key Concepts

### Authority shortcut

**Definition**: People often defer to recognized authority because it is usually efficient. Cialdini discusses automatic obedience in the authority chapter.

### Title signal

**Definition**: Titles can generate deference even when detached from real expertise.

### Clothing signal

**Definition**: Uniforms and professional attire can imply authority.

### Trappings signal

**Definition**: Status objects and institutional symbols can transfer credibility.

### Relevance gap

**Definition**: A real credential can still be irrelevant to the specific claim.

## How It Relates To Other Influence Principles

- Reciprocity asks whether value creates obligation.
- Commitment asks whether prior action creates consistency pressure.
- Social proof asks whether other people's behavior is being used as evidence.
- Liking asks whether relationship cues are affecting trust.
- Authority asks whether credentials or symbols create deference.
- Scarcity asks whether loss or limited availability creates urgency.
- Contrast asks whether sequence changes perceived value.

## Common Misconceptions

- **Myth**: Persuasion is unethical by default. **Reality**: Persuasion is useful when it gives truthful, decision-relevant evidence.
- **Myth**: If a cue works, it should be used. **Reality**: A cue should be used only when it accurately represents the situation.
- **Myth**: Removing pressure means weakening the message. **Reality**: Honest specificity often increases trust and long-term conversion quality.
