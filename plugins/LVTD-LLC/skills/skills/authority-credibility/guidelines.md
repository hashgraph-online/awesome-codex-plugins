# Authority Credibility Guidelines

Quick routing for authority credibility tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/authority/knowledge.md |
| Create or rewrite copy | references/authority/rules.md, references/authority/examples.md |
| Audit an existing surface | references/authority/smells.md, references/authority/checklist.md |
| Run the full workflow | workflows/verify-credibility-signals.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/authority/smells.md |
| The evidence behind a claim is unclear | references/authority/rules.md |
| The user needs a practical rewrite | references/authority/examples.md |
| The task has several steps or stakeholders | workflows/verify-credibility-signals.md |
