# Draft LinkedIn Post Workflow

Use this workflow when the user asks for a LinkedIn post draft, rewrite, or
post audit.

## Steps

### Step 1: Capture Inputs

- [ ] Audience.
- [ ] Topic territory.
- [ ] Author point of view or experience.
- [ ] Desired reader action.
- [ ] Proof, example, news item, or asset.

### Step 2: Select The Post Job

- [ ] Teach.
- [ ] Interpret news.
- [ ] Share a personal lesson.
- [ ] Start a discussion.
- [ ] Promote an article, document, video, or event.
- [ ] Announce with reader relevance.

### Step 3: Draft

- [ ] Open with the useful tension or takeaway.
- [ ] Keep paragraphs short.
- [ ] Add interpretation or proof.
- [ ] Include a natural comment prompt or next action.
- [ ] Keep within the post limit or route to article/newsletter.

### Step 4: Polish For Platform Fit

- [ ] Add 2-5 relevant hashtags if useful.
- [ ] Add mentions only when justified.
- [ ] Add alt text guidance for images.
- [ ] Add captions guidance for video.
- [ ] Remove fake engagement, irrelevant tags, and manipulative asks.

## Exit Criteria

- [ ] Draft is ready to paste or schedule.
- [ ] Publishing notes explain hashtags, mentions, links, and media.
- [ ] Any current-platform assumptions are either checked or flagged.
