# Repurpose Post To Long-Form Workflow

Use this workflow when a LinkedIn post, thread, talk, or draft should become an
article or newsletter edition.

## Steps

### Step 1: Decide The Format

- [ ] Keep as post if the idea is complete in short form.
- [ ] Use article if the idea needs depth or durable profile value.
- [ ] Use newsletter if this is part of a recurring promise.

### Step 2: Expand The Spine

- [ ] Reader problem.
- [ ] Common mistake.
- [ ] Framework or point of view.
- [ ] Example.
- [ ] Next step.

### Step 3: Add Distribution

- [ ] Write title.
- [ ] Write teaser post.
- [ ] Add hashtags only if relevant.
- [ ] Add bio or profile CTA.
- [ ] Define comment prompt.

### Step 4: Check Platform Details

- [ ] Confirm current article/newsletter availability if needed.
- [ ] Confirm Page vs member publishing path if relevant.
- [ ] Confirm analytics to review after publishing.

## Exit Criteria

- [ ] Long-form draft teaches or explains instead of advertising.
- [ ] Teaser post can distribute it in the feed.
- [ ] Cadence and next step are realistic.
