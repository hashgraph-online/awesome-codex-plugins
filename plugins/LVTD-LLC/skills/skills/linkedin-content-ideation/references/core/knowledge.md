# LinkedIn Content Ideation Knowledge

## Core Idea

The book treats LinkedIn growth as a cycle: generate ideas, prioritize them,
test them, analyze the response, then generate better ideas. For content, this
means every post topic is a hypothesis about what the audience will value and
what the author should become known for.

## Idea Sources

| Source | How To Use It |
|--------|---------------|
| Audience questions | Turn repeated questions into explanatory posts, checklists, or contrarian answers. |
| Industry news | Add "what this means for [audience]" rather than reposting the news. |
| Hashtag search | Find active conversations and unanswered angles inside the author's topic. |
| Quora and question tools | Harvest real wording for problem-aware post topics. |
| Twitter/X or other social search | Find compressed arguments, trends, and debated opinions to translate for LinkedIn. |
| Comments | Turn objections and follow-up questions into new posts. |
| Company or project material | Convert launches, milestones, research, and internal lessons into audience-relevant content. |
| Personal proof | Use events, books, screenshots, artifacts, and photos to connect the author with a topic. |

## Idea Quality Criteria

Good LinkedIn content ideas have:

- A specific audience.
- A clear reader problem, curiosity, belief, or decision.
- A visible author point of view.
- A reason this idea should be posted now.
- A format that suits the idea: text, image, document, video, article, poll, or
  newsletter.
- A measurable response: comments, saves, profile visits, newsletter follows,
  link clicks, or qualified conversations.

## Tool Caution

The book names specific tools, but tool availability, pricing, and LinkedIn
behavior change. Treat tools as categories:

- Question mining.
- Social search.
- Content curation.
- Draft preview.
- Scheduling.
- Analytics.
- Image preparation.

Recommend current tools only after checking whether they still exist and fit the
user's workflow.
