# LinkedIn Content Ideation Examples

## Backlog Row Template

| Field | Example |
|-------|---------|
| Idea | Why teams over-measure impressions and under-measure qualified comments |
| Source | Comment thread from last week's analytics post |
| Post Job | Teach / reframe |
| Why It Fits | Audience wants better B2B content metrics; author owns content strategy |
| Asset | Screenshot of metric dashboard or simple table |
| Priority | High |

## Question Mining Prompts

- What does [audience] misunderstand about [topic]?
- What decision does [audience] need to make before [event]?
- What changed in [industry] that most people have not adapted to?
- What advice is popular but harmful for [audience]?
- What did the author learn the hard way about [topic]?

## Idea Families

### Industry News

```markdown
News: [event]
Audience implication: [why it matters]
Author POV: [what most people miss]
Post format: short text post or document
```

### Personal Brand Proof

```markdown
Proof: [event, artifact, photo, project, result]
Topic association: [what the author wants to own]
Reader value: [lesson or useful framing]
Post format: image + short lesson
```

### Comment-Led Idea

```markdown
Trigger comment: [objection/question]
New post angle: [answer or reframing]
Expected response: [discussion, saves, follow-up]
```

## Prioritization Matrix

| Score | Meaning |
|-------|---------|
| 3 | Strong audience pain, strong author credibility, clear post angle. |
| 2 | Useful idea but needs sharper proof or format. |
| 1 | Vague, generic, off-territory, or hard to execute. |
