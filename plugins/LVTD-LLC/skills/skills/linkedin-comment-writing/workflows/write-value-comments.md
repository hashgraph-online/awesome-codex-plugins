# Write Value Comments Workflow

Use this workflow when the user wants comments for specific LinkedIn posts or a
repeatable comment routine.

## Steps

### Step 1: Understand The Thread

- [ ] Identify the post's main claim.
- [ ] Identify the audience.
- [ ] Identify the existing comment tone.
- [ ] Decide whether joining is worthwhile.

### Step 2: Choose Comment Job

- [ ] Extend.
- [ ] Add example.
- [ ] Clarify.
- [ ] Respectfully disagree.
- [ ] Ask better question.
- [ ] Summarize implication.

### Step 3: Draft Options

- [ ] Write 2-3 comments with different angles.
- [ ] Keep each comment natural and specific.
- [ ] Remove product pitches unless directly relevant.
- [ ] Remove generic praise.

### Step 4: Follow Up

- [ ] If the author replies, continue the useful thread.
- [ ] If someone challenges the comment, answer the substance.
- [ ] If the thread becomes low-quality, stop.

## Exit Criteria

- [ ] Comment adds value to the original post.
- [ ] Comment matches the user's professional voice.
- [ ] No fake engagement or spam signal remains.
