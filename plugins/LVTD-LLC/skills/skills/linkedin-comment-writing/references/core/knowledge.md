# LinkedIn Comment Writing Knowledge

## Core Idea

The book treats comments as one of the strongest visibility mechanisms on
LinkedIn because they place the author inside existing conversations and can
keep the author's own posts active. The useful version of this tactic is not
volume for its own sake; it is adding context, examples, questions, and
thoughtful replies where the target audience already pays attention.

## Comment Jobs

| Job | Use When | Shape |
|-----|----------|-------|
| Extend | The post is right but incomplete. | "Another case where this shows up is..." |
| Add example | The thread is abstract. | Concrete story, metric, customer case, or pattern. |
| Clarify | The post is useful but ambiguous. | Define a term or separate two similar ideas. |
| Respectfully disagree | The post overstates or misses an exception. | Agree with intent, name the exception, explain why. |
| Ask better question | The thread needs depth. | Ask about mechanism, trade-off, or evidence. |
| Summarize implication | The post has many reactions but no next step. | "The practical takeaway for [audience] is..." |

## Comment Cadence

A practical cadence is to find a small number of relevant posts through
hashtags, authors, or topic searches and write fewer, stronger comments. The
book gives an example of commenting on multiple hashtag-relevant posts over
several weeks, but the durable principle is consistency plus relevance, not a
fixed quota.
