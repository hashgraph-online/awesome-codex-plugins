# Visual Docs Diagrams Guidelines

Load the minimum files needed for the task.

## By Task

| What you're doing | Load these files |
|-------------------|------------------|
| Designing a technical visual | `references/core/knowledge.md`, `workflows/design-technical-visual.md` |
| Reviewing screenshots | `references/core/knowledge.md` |
| Choosing a diagram type | `references/core/knowledge.md` |
| Checking accessibility or performance | `references/core/knowledge.md` |
| Deciding whether to use video | `references/core/knowledge.md` |

## By Problem

| If you notice... | Load these files |
|------------------|------------------|
| A visual repeats text without adding comprehension | `references/core/knowledge.md` |
| A diagram mixes architecture, sequence, and UI in one image | `workflows/design-technical-visual.md` |
| Critical copyable values appear only in a screenshot | `references/core/knowledge.md` |
| A visual has no source file or update trigger | `references/core/knowledge.md` |
