# Influence Audit Guidelines

Quick routing for influence audit tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/audit/knowledge.md |
| Create or rewrite copy | references/audit/rules.md, references/audit/examples.md |
| Audit an existing surface | references/audit/smells.md, references/audit/checklist.md |
| Run the full workflow | workflows/audit-influence-risk.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/audit/smells.md |
| The evidence behind a claim is unclear | references/audit/rules.md |
| The user needs a practical rewrite | references/audit/examples.md |
| The task has several steps or stakeholders | workflows/audit-influence-risk.md |

## Decision Tree

```
Start
|- Need concepts? -> references/audit/knowledge.md
|- Need to create? -> references/audit/rules.md + examples.md
|- Need to audit? -> references/audit/smells.md + checklist.md
\- Need repeatable process? -> workflows/audit-influence-risk.md
```
