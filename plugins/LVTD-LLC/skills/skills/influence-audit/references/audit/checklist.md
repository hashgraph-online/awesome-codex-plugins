# Influence Audit Checklist

Use this checklist before finalizing advice, copy, or an audit.

## Evidence

- [ ] Every proof claim has source, date, and scope.
- [ ] Every scarcity claim maps to a real constraint.
- [ ] Every authority signal states the credential's relevance and limits.
- [ ] Every reciprocal offer can be declined without penalty.
- [ ] The user can still compare, pause, and say no.

## Red Flags

- [ ] Evidence-free superlatives
- [ ] Vanishing opportunity without operational reason
- [ ] Borrowed authority
- [ ] Obligation trap

## Final Pass

- [ ] The recommendation names the relevant Influence principle.
- [ ] The recommendation distinguishes real evidence from pressure.
- [ ] The rewrite is specific enough to use immediately.
- [ ] The user can still pause, compare, decline, or ask questions.

## Source Traceability

- [ ] Attribute book-derived concepts without relying on unpackaged local extraction paths.
- [ ] Paraphrase book content rather than copying long passages.
