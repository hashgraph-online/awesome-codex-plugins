# Scarcity Urgency Guidelines

Quick routing for scarcity urgency tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/scarcity/knowledge.md |
| Create or rewrite copy | references/scarcity/rules.md, references/scarcity/examples.md |
| Audit an existing surface | references/scarcity/smells.md, references/scarcity/checklist.md |
| Run the full workflow | workflows/design-honest-urgency.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/scarcity/smells.md |
| The evidence behind a claim is unclear | references/scarcity/rules.md |
| The user needs a practical rewrite | references/scarcity/examples.md |
| The task has several steps or stakeholders | workflows/design-honest-urgency.md |
