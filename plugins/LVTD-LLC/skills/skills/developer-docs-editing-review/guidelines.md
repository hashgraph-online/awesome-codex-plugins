## By Problem

| If you notice... | Load these files |
|------------------|------------------|
| Procedures have not been run or verified | `references/core/knowledge.md` |
| The doc has TODOs, missing prerequisites, or version gaps | `workflows/edit-developer-doc.md` |
| A migration, upgrade, data operation, or production change needs validation | `workflows/verify-technical-doc.md` |
| Feedback is vague or preference-based | `references/core/knowledge.md` |
| Reviewers disagree | `references/core/knowledge.md` |
| Required reviewers are late, missing, or gave shallow feedback | `workflows/verify-technical-doc.md` |
