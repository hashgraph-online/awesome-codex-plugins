# Ethical Persuasion Checklist

Use this checklist before finalizing advice, copy, or an audit.

## Evidence

- [ ] The ask is specific and easy to understand.
- [ ] The main proof is true, sourced, and relevant.
- [ ] The user benefit is concrete.
- [ ] The user has enough context to say no.
- [ ] The copy avoids invented or exaggerated cues.

## Red Flags

- [ ] Principle pileup
- [ ] Missing user's reason
- [ ] Hidden constraint

## Final Pass

- [ ] The recommendation names the relevant Influence principle.
- [ ] The recommendation distinguishes real evidence from pressure.
- [ ] The rewrite is specific enough to use immediately.
- [ ] The user can still pause, compare, decline, or ask questions.

## Source Traceability

- [ ] Attribute book-derived concepts without relying on unpackaged local extraction paths.
- [ ] Paraphrase book content rather than copying long passages.
