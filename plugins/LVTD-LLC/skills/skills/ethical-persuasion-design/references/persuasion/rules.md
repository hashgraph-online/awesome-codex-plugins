# Ethical Persuasion Rules

Rules for applying ethical persuasion without counterfeit evidence or coercive pressure.

## Core Rules

### 1. Start with the user's decision

Define what the user is trying to decide and what information would genuinely help.

### 2. Use real proof before polished claims

Prefer specific evidence, numbers, named examples, and constraints over broad persuasion language.

### 3. Use one primary principle

Choose the most relevant Influence principle for the situation instead of stacking every available cue.

### 4. State tradeoffs plainly

If there is a cost, limit, eligibility condition, or risk, surface it near the ask.

## Guidelines

- Use reciprocity for genuine education or samples, not obligation.
- Use commitment for authentic user intent, not sunk-cost traps.
- Use social proof when the audience and proof group are meaningfully similar.
- Use authority only when the credential is relevant to the claim.
- Use scarcity only when the constraint is real.

## Ethical Boundaries

- Use only honest evidence. Do not invent popularity, scarcity, credentials, endorsements, or social connection.
- Separate helping a good decision from pushing a shortcut response. If the cue is counterfeit, treat it as a red flag.
- When rewriting, preserve user agency: add context, alternatives, and enough time to decide when stakes are meaningful.

## Exceptions

- **Parody, critique, or analysis**: Describe manipulative tactics when the purpose is detection, education, or safer rewriting.
- **Real constraints**: Scarcity, authority, popularity, or reciprocity cues are acceptable when accurate, scoped, and relevant.
- **Tiny low-stakes choices**: Lighter disclosure may be fine, but never fabricate evidence.
