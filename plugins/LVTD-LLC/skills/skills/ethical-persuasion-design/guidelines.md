# Ethical Persuasion Design Guidelines

Quick routing for ethical persuasion design tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/persuasion/knowledge.md |
| Create or rewrite copy | references/persuasion/rules.md, references/persuasion/examples.md |
| Audit an existing surface | references/persuasion/smells.md, references/persuasion/checklist.md |
| Run the full workflow | workflows/design-persuasive-request.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/persuasion/smells.md |
| The evidence behind a claim is unclear | references/persuasion/rules.md |
| The user needs a practical rewrite | references/persuasion/examples.md |
| The task has several steps or stakeholders | workflows/design-persuasive-request.md |

## Decision Tree

```
Start
|- Need concepts? -> references/persuasion/knowledge.md
|- Need to create? -> references/persuasion/rules.md + examples.md
|- Need to audit? -> references/persuasion/smells.md + checklist.md
\- Need repeatable process? -> workflows/design-persuasive-request.md
```
