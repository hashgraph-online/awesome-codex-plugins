## Decision Tree

```text
What blocks adoption?
|- Trust in data -> prove with small useful tests
|- Fear of delay -> reduce process weight
|- Fear of failure -> reframe as learning
|- Tool friction -> improve templates/support
`- Demand growth -> create cadence and governance
```
