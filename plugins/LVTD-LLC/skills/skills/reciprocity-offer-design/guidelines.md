# Reciprocity Offer Design Guidelines

Quick routing for reciprocity offer design tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/reciprocity/knowledge.md |
| Create or rewrite copy | references/reciprocity/rules.md, references/reciprocity/examples.md |
