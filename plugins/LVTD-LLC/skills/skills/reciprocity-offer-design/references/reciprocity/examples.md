# Reciprocity Examples

Examples are paraphrased applications of the principle, not book excerpts.

## Bad and Better Examples

### Lead magnet

Bad: 'I gave you this guide, so book now.' Better: 'Use the guide either way; if you want help applying it, here are the next options.'

### Free sample

Bad: A sample is handed over and immediately paired with social pressure. Better: A sample is offered with explicit permission to pass.

### Concession

Bad: Start with a fake expensive package to make the real one feel cheap. Better: Present the best-fit package first and explain alternatives.

## Rewrite Pattern

1. Identify the active cue.
2. Remove unsupported claims or coercive framing.
3. Add the real evidence, constraint, or user benefit.
4. Make the next step clear but optional.

## Mini Review Example

**Input**: A message uses pressure but does not show evidence.

**Output**:
- Principle: Reciprocity
- Risk: unsupported cue may create a shortcut response.
- Rewrite: state the real reason, proof, or constraint; remove fake pressure.
