# Contrast Framing Guidelines

Quick routing for contrast framing tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/contrast/knowledge.md |
| Create or rewrite copy | references/contrast/rules.md, references/contrast/examples.md |
| Audit an existing surface | references/contrast/smells.md, references/contrast/checklist.md |
| Run the full workflow | workflows/sequence-comparisons.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/contrast/smells.md |
| The evidence behind a claim is unclear | references/contrast/rules.md |
| The user needs a practical rewrite | references/contrast/examples.md |
| The task has several steps or stakeholders | workflows/sequence-comparisons.md |

## Decision Tree

```
Start
|- Need concepts? -> references/contrast/knowledge.md
|- Need to create? -> references/contrast/rules.md + examples.md
|- Need to audit? -> references/contrast/smells.md + checklist.md
\- Need repeatable process? -> workflows/sequence-comparisons.md
```
