# Social Proof Evidence Guidelines

Quick routing for social proof evidence tasks. Load only the files needed for the current request.

## By Task

| What you are doing | Load these files |
| --- | --- |
| Understand the principle | references/social-proof/knowledge.md |
| Create or rewrite copy | references/social-proof/rules.md, references/social-proof/examples.md |
| Audit an existing surface | references/social-proof/smells.md, references/social-proof/checklist.md |
| Run the full workflow | workflows/choose-social-proof.md |

## By Problem or Symptom

| If you notice | Load these files |
| --- | --- |
| The copy feels pushy or manipulative | references/social-proof/smells.md |
| The evidence behind a claim is unclear | references/social-proof/rules.md |
| The user needs a practical rewrite | references/social-proof/examples.md |
| The task has several steps or stakeholders | workflows/choose-social-proof.md |

## Decision Tree

```
Start
|- Need concepts? -> references/social-proof/knowledge.md
|- Need to create? -> references/social-proof/rules.md + examples.md
|- Need to audit? -> references/social-proof/smells.md + checklist.md
\- Need repeatable process? -> workflows/choose-social-proof.md
```
