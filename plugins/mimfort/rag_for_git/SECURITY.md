# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities privately to the maintainers.

**Do not open a public issue.** Instead, email the maintainers directly.

We will respond within 5 business days and work with you on a fix and coordinated disclosure timeline.

## Scope

- The reviewer MCP server (`reviewer-mcp`) and its tools
- The CLI entry points
- The RAG pipeline and code-graph construction
- Dependencies and supply chain

## Out of Scope

- Issues already tracked as public bugs or feature requests
- Third-party services the project integrates with
