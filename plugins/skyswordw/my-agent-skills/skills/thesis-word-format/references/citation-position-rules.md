# Citation Position Rules

Use the sequential numeric system for this thesis.

Current standard context:

- GB/T 7714-2015 remains the active issued standard until GB/T 7714-2025 takes effect on 2026-07-01 unless the school requests earlier adoption.
- The 2025 public draft keeps the sequential numeric system: references are numbered by first appearance in the text and the numbers are placed in square brackets.
- The draft examples show clause/sentence evidence markers after the cited statement, and author-led examples such as a named author followed by `[n]`.
- Some university thesis rules explicitly set ordinary citation markers as superscript, but leave markers baseline when the citation marker is an effective sentence component, e.g. `由文献[8,10-13]可知`. This project uses the user's stricter visual rule: in-text numeric markers are superscript even inside `文献[n]`.
- For this Chinese thesis, prefer `文献[n]...` when the citation is the sentence subject because English-author-led openings are usually English-paper style.

Practical editing rules:

- Sentence-final or clause-final evidence: `......机制[12]。` -> `[12]` as superscript.
- Narrative citation at the sentence opening: `文献[12]指出......` -> keep `文献` baseline and set `[12]` as superscript.
- Author retained for meaning: `Sun等在文献[12]中指出......` is acceptable; do not use bare `Sun等[12]指出......` or `Chen等[12]提出......` as a Chinese sentence/clause opener unless the user requests English-paper phrasing. The normal project rewrite is `文献[12]指出/提出......`. Chinese author-led wording such as `张三等[n]认为...` may be allowed by some school templates, but this project should prefer `文献[n]...` for Chinese narrative openings unless the author identity matters.
- Multiple references: prefer `[3,7-9]` in one marker.
- Page-specific repeated citations: if page numbers are needed, place page information outside the brackets per the active standard/examples and verify with the institution requirement.
