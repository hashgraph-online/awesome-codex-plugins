# BGS Modding Superpowers

Mirrored bundle of [BB-84C/bgs-modding-superpowers](https://github.com/BB-84C/bgs-modding-superpowers) for the awesome-codex-plugins marketplace.

Agentic superpowers for Bethesda Game Studio modpack curation:

- MCP-driven xEdit conflict audit
- MO2 control plane (Python broker + TypeScript MCP server)
- Skills for first-time setup, dev-log maintenance, and release-changelog generation
- BA2/BSA archive inspection, Papyrus compile/decompile, LLM-assisted plugin translation

See the upstream repository for full documentation, install instructions, and the per-game knowledge-base packs.
