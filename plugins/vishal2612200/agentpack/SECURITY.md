# Security Policy

Report security issues privately through GitHub security advisories for:

https://github.com/vishal2612200/agentpack/security/advisories/new

Security-relevant areas include local source scanning, generated context artifacts, MCP access to repo context, installer-written hooks, and release packaging.

AgentPack does not upload source code for scan, route, pack, stats, or benchmark commands. Generated files live under `.agentpack/`; review them before sharing outside your machine.
