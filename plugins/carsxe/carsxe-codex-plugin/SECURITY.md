# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a Vulnerability

To report a security vulnerability, please email security@carsxe.com.

Do **not** open a public GitHub issue for security vulnerabilities.

We aim to respond within 48 hours and will work with you to resolve the issue promptly.

## Scope

This policy covers the CarsXE Codex Plugin source code at https://github.com/carsxe/carsxe-codex-plugin.

It does not cover the CarsXE API itself — report API security issues directly to security@carsxe.com.
